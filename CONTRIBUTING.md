# Contributing to @idapt/sdk

Thanks for your interest in improving the idapt SDK!

> **This repository is a read-only mirror** of idapt's development monorepo. The
> source is republished on each release, so direct pushes here are overwritten
> by the next sync. That means:
>
> - **Issues & ideas** are very welcome — please open one here.
> - **Code changes**: open an issue describing the change first. Accepted
>   patches are applied upstream and ship in the next release.

## License & sign-off

`@idapt/sdk` is licensed under **Apache-2.0** (see [LICENSE](./LICENSE)).
Contributions are accepted under the same license — inbound = outbound, per
Apache-2.0 §5. Please sign off your commits under the
[Developer Certificate of Origin](https://developercertificate.org/):

```bash
git commit -s -m "your message"
```

The `-s` adds a `Signed-off-by` line certifying you have the right to submit the
contribution.

## Security

Please report vulnerabilities privately — see [SECURITY.md](./SECURITY.md). Do
not open public issues for security problems.
